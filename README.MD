# My dotfiles

# Prerequisites

## Mac OS
  * [Brew](https://brew.sh/)
  * Git
     ```sh
      brew install git
    ```
# Common
  * [Vim Vundle](https://github.com/VundleVim/Vundle.vim#quick-start)

# Installation
```sh
cd
git clone --recursive git@github.com:alxzoomer/dotfiles.git
cd dotfiles
chmod +x setup.sh
./setup.sh
```
